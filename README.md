### 🧬 [醫療廢棄物暨資源管理系統整合應用 —— 獲獎作品](https://github.com/zhilingliu-work/waste_system_django)
* **專案概述：** 本系統為醫療機構設計之智慧化醫療廢棄物數位追蹤與管理平台，基於 Django 5.1.6 框架開發，整合 IoT 硬體感測器導入數位追蹤工作流，全功能聚焦於異常即時警報與智慧審計機制，為現代化智慧醫院提供全方位技術保障。
* **技術亮點：** 整合藍牙電子磅秤與 QR Code 列印，實現現場實體與數據端精準數位勾稽；設計嚴格的 RBAC 權限分流機制，劃分 6 大身分組管理體系；配置主動式異常警報系統，結合數據趨勢視覺化與自動化報表匯出，徹底杜絕醫療廢棄物非法外流風險。

### 🧬 [任務排程最佳化研究 —— 遺傳演算法 (GA) 核心機制與應用](https://github.com/zhilingliu-work/Genetic-Algorithm-Research)
* **專案概述：** 針對平行運算系統中屬於 NP-hard 的 DAG 任務排程問題，利用 Python 實作啟發式最佳化求解框架。
* **技術亮點：** 設計 `(ss, ms)` 雙元件染色體編碼以完美符合拓撲排序限制；導入**三階段動態突變率**（0.27 ➔ 0.12 ➔ 0.02）優化全域搜尋，並加入「連續 5 代未改善則強制突變」之**防陷落機制**，結合 $60\%$ 的**激進精英保留策略**，顯著提升演算法收斂效率與求解品質。
